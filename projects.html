Projects!

    Holding Hand

    Android application designed to serve as a safe walking tool, you would need to have a finger on the screen at all time while walking, after you are done you would need to put a pin, but if you dont it will send a message to your emergency contacts. Uses the Google Maps API, and the standard SMS delivery system for Android.

    Simon Says

    Scavenger  Hunt  game  for  Android  using  the  Clarifai  image  recognition  API. A list of items is given and the user needs to take pictures of te items in order to win the game. Contains a multiplayer feature were it measures the time that it took you to take the pictures of the item list.

    Rate ISU

    Chrome extension that integrates Rate my Professor to Iowa State University schedule planner. Uses data scrapping customized to ratemyprofessor.com; created to facilitate the process of registering for classes, saving students time by showing the rating of the professor right next to their names.

    Follow Me Vest

    Night time illumination jogging vest powered by Arduino and C embedded code. Designed for night time joggers, it contains flashlights and turning signals that will help vehicles visualize the person excersicing.
